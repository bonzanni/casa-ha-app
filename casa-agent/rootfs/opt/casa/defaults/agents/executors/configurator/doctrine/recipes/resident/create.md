# Recipe: create a new resident — RETIRED

Residents are fixed: assistant, butler, concierge. New residents cannot be
created — the roster is part of the platform contract, and the loader only
compiles persona bindings for the fixed roles.

- New capability the household needs → that is a SPECIALIST: follow
  `recipes/specialist/install.md`.
- Different identity/voice for an existing resident → that is a PERSONA:
  follow `recipes/persona/install.md` + `recipes/persona/apply.md`.
